# What is a for loop?
<br>
A *for loop* in python repeats a block of code for each item in a *sequence* (like a list, tuple, string, range, or dictionary keys).
<br>
Its the most common way to iterate (go though items one by one) in python.
<br>
<br>
**Key Idea**: "for each item in the collection, do something".
<br>

## Basic syntax
<br>
'''python
for item in sequence:
#code to run for each item (indented with 4 spaces)
print(item)       
```
